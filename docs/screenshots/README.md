# Screenshot Generation

Screenshots in this directory are auto-generated by Playwright and used in the
[Bundle Editor Guide](../../BUNDLE-EDITOR-GUIDE.md).

## Regenerating

From the project root:

```bash
npm run screenshots --prefix monitor
```

This runs against **staging** by default. To target a different environment:

```bash
SCREENSHOT_BASE_URL=https://menu2.highhopesma.com npm run screenshots --prefix monitor
```

## Requirements

- Playwright + Chromium installed (same setup as the e2e monitor)
- The target environment must be running and accessible
- A test bundle is created and cleaned up automatically during the run

## Viewport

All screenshots use an iPad Air viewport (1180x820) at 2x device scale for
Retina-quality images.
